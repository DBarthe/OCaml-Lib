Module PriorityQueue

module PriorityQueue: sig .. end
A functionnal priorityQueue implemented with a binomial heap.
Author(s): Barthelemy Delemotte
type 'a t
The priority queue type.
val create : ('a -> 'a -> int) -> 'a t
create compare_fun returns a new empty priority queue. The function compare_fun is used to determine how to sort elements in the queue. (compare_fun a b < 0 means a have priority on b).
val is_empty : 'a t -> bool
is_empty q returns true if q is empty.
val length : 'a t -> int
length q returns the number of elements in q.
val top : 'a t -> 'a
top q returns the minimum element contained in q according to the comparison function compare_fun (given during construction).
Raises Failure when q is empty.
val push : 'a t -> 'a -> 'a t
push q elt adds the element elt to q.
val pop : 'a t -> 'a t
pop q removes the minimum element of q.
Raises Failure when q is empty.
val from_list : ('a -> 'a -> int) -> 'a list -> 'a t
from_list compare_fun l creates a new priority queue with elements of l.
val to_list : 'a t -> 'a list
to_list q returns a sorted list composed by elements of q.
val merge : 'a t -> 'a t -> 'a t
merge q1 q2 creates a new priority queue by merging q1 and q2. The function compare_fun of q1 q2 should be the same. But this could not be verified. If they are not, the result is unspecified.
